LED 사용 방법

LED는 전류가 흐르면 빛을 내는 전자 부품입니다. 아두이노, ESP32 같은 보드에서 상태 표시등, 경고등, 조명, 신호 출력 장치 등에 가장 기본적으로 사용됩니다.

LED란?

LED는 Light Emitting Diode의 줄임말로, 전류가 한쪽 방향으로 흐를 때 빛을 내는 다이오드입니다. 극성이 있기 때문에 긴 다리와 짧은 다리를 구분해서 연결해야 합니다.

LED 다리 구분

  • 긴 다리: +극, 애노드
  • 짧은 다리: -극, 캐소드
  • 평평한 부분이 있는 쪽: 보통 -극입니다.
LED는 반드시 저항과 함께 사용하는 것이 좋습니다.
저항 없이 연결하면 LED나 보드 핀이 손상될 수 있습니다.

아두이노 연결 방법

LED 부품 아두이노 연결 역할
LED 긴 다리 저항을 거쳐 D13 제어 신호 입력
LED 짧은 다리 GND 접지
저항 220Ω ~ 330Ω 전류 제한
LED는 방향이 반대로 연결되면 켜지지 않습니다.

LED 깜빡이기 예시 코드

아래 코드는 LED를 1초 동안 켜고, 1초 동안 끄는 동작을 반복합니다.

Arduino Code 
#include <Arduino.h>

const int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(1000);

  digitalWrite(ledPin, LOW);
  delay(1000);
}

LED 밝기 조절 예시 코드

PWM 핀을 사용하면 LED 밝기를 조절할 수 있습니다. 아두이노 UNO에서는 보통 3, 5, 6, 9, 10, 11번 핀이 PWM을 지원합니다.

Arduino PWM Code 
#include <Arduino.h>

const int ledPin = 9;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  analogWrite(ledPin, 50);
  delay(1000);

  analogWrite(ledPin, 150);
  delay(1000);

  analogWrite(ledPin, 255);
  delay(1000);

  analogWrite(ledPin, 0);
  delay(1000);
}

사용할 때 주의할 점

  • LED는 극성이 있으므로 긴 다리와 짧은 다리를 구분해야 합니다.
  • LED에는 220Ω 또는 330Ω 정도의 저항을 함께 연결하는 것이 좋습니다.
  • 디지털 출력은 LED를 켜고 끄는 데 사용합니다.
  • PWM 출력은 LED 밝기를 조절할 때 사용합니다.
  • ESP32에서는 PWM 제어 방식이 아두이노 UNO와 다를 수 있습니다.
