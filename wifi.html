Wi-Fi 통신 모듈 사용 방법

Wi-Fi 통신 모듈은 아두이노 같은 보드가 인터넷이나 같은 네트워크에 연결된 장치와 데이터를 주고받을 수 있게 해주는 모듈입니다. 대표적으로 ESP8266 ESP-01, ESP8266 NodeMCU, ESP32 등이 사용됩니다.

Wi-Fi 통신 모듈이란?

Wi-Fi 통신 모듈은 무선 네트워크를 이용해 데이터를 전송하는 부품입니다. 아두이노 UNO처럼 Wi-Fi 기능이 없는 보드는 ESP8266 같은 모듈을 연결해서 인터넷 통신 기능을 추가할 수 있습니다.

대표적인 Wi-Fi 모듈

  • ESP8266 ESP-01: 아두이노에 연결해서 Wi-Fi 기능을 추가할 때 자주 사용합니다.
  • NodeMCU ESP8266: ESP8266을 개발보드 형태로 만든 보드입니다.
  • ESP32: Wi-Fi와 블루투스가 내장된 고성능 보드입니다.
ESP8266 ESP-01은 보통 UART 시리얼 통신으로 아두이노와 연결합니다.
AT 명령어를 보내서 Wi-Fi 연결, 서버 접속, 데이터 전송 등을 제어할 수 있습니다.

동작 원리

  • 아두이노가 ESP8266 모듈에 AT 명령어를 보냅니다.
  • ESP8266은 명령어에 따라 Wi-Fi 공유기에 접속합니다.
  • 네트워크에 연결된 뒤 서버나 웹사이트와 데이터를 주고받습니다.
  • 받은 데이터를 다시 아두이노로 전달할 수 있습니다.

아두이노 연결 방법

아래 연결 예시는 ESP8266 ESP-01 모듈을 기준으로 합니다.

ESP8266 ESP-01 핀 아두이노 연결 핀 역할
VCC 3.3V 전원 공급
GND GND 접지
TX D2 ESP8266이 보내는 데이터 수신
RX D3 ESP8266으로 데이터 전송
CH_PD / EN 3.3V 모듈 활성화
RST 3.3V 또는 미연결 리셋 제어
ESP8266은 3.3V 전원을 사용합니다. 5V를 직접 연결하면 모듈이 손상될 수 있습니다.

아두이노 예시 코드

아래 코드는 아두이노에서 ESP8266 ESP-01 모듈로 AT 명령어를 보내 Wi-Fi 연결을 시도하는 예시입니다.

Arduino Code 
#include <Arduino.h>
#include <SoftwareSerial.h>

SoftwareSerial esp8266(2, 3);

String ssid = "YOUR_WIFI_NAME";
String password = "YOUR_WIFI_PASSWORD";

void sendCommand(String command, int delayTime) {
  esp8266.println(command);
  delay(delayTime);

  while (esp8266.available()) {
    Serial.write(esp8266.read());
  }
}

void setup() {
  Serial.begin(9600);
  esp8266.begin(9600);

  Serial.println("ESP8266 Wi-Fi Module Start");

  sendCommand("AT", 1000);
  sendCommand("AT+CWMODE=1", 1000);

  String wifiCommand = "AT+CWJAP=\"" + ssid + "\",\"" + password + "\"";
  sendCommand(wifiCommand, 6000);

  sendCommand("AT+CIFSR", 1000);
}

void loop() {
  if (esp8266.available()) {
    Serial.write(esp8266.read());
  }

  if (Serial.available()) {
    esp8266.write(Serial.read());
  }
}

코드 설명

  • SoftwareSerial esp8266(2, 3): D2를 RX, D3을 TX처럼 사용합니다.
  • AT: ESP8266 모듈이 정상적으로 응답하는지 확인합니다.
  • AT+CWMODE=1: Wi-Fi 모듈을 공유기에 접속하는 Station 모드로 설정합니다.
  • AT+CWJAP: Wi-Fi 이름과 비밀번호를 사용해 공유기에 접속합니다.
  • AT+CIFSR: Wi-Fi 연결 후 IP 주소를 확인합니다.

사용할 때 주의할 점

  • ESP8266 ESP-01은 반드시 3.3V로 전원을 공급해야 합니다.
  • 아두이노의 3.3V 핀은 전류가 부족할 수 있어 별도 3.3V 전원 모듈 사용을 권장합니다.
  • TX와 RX는 서로 교차해서 연결해야 합니다.
  • ESP8266의 RX 핀은 3.3V 입력이므로 아두이노 5V 신호를 직접 넣지 않는 것이 안전합니다.
  • Wi-Fi 이름과 비밀번호는 코드에서 직접 수정해야 합니다.
