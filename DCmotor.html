DC 모터 사용 방법

DC 모터는 전류를 흘려주면 회전하는 가장 기본적인 모터입니다. 아두이노, ESP32 같은 보드에서 자동차 바퀴, 팬, 컨베이어 벨트, 로봇 구동 장치 등에 자주 사용됩니다.

DC 모터란?

DC 모터는 직류 전원을 사용해 회전하는 모터입니다. 전원의 극성을 바꾸면 회전 방향이 바뀌고, 전압이나 PWM 신호를 조절하면 회전 속도를 어느 정도 제어할 수 있습니다.

동작 원리

  • 모터에 전류가 흐르면 내부 코일에 자기장이 생깁니다.
  • 코일의 자기장과 자석의 자기장이 서로 힘을 주고받습니다.
  • 그 힘 때문에 모터의 축이 회전합니다.
  • 전류 방향을 바꾸면 모터의 회전 방향도 바뀝니다.
DC 모터는 아두이노 핀에 직접 연결하면 안 됩니다.
모터는 많은 전류를 사용하기 때문에 모터 드라이버를 통해 제어해야 합니다.

모터 드라이버가 필요한 이유

아두이노의 디지털 핀은 작은 전류만 출력할 수 있습니다. 하지만 DC 모터는 회전할 때 더 많은 전류가 필요합니다. 따라서 아두이노는 모터에 직접 전원을 주는 것이 아니라, 모터 드라이버에게 방향과 속도 제어 신호를 보내는 역할을 합니다.

대표적인 모터 드라이버

  • L298N: DC 모터 2개 제어 가능
  • L293D: 소형 DC 모터 제어에 자주 사용
  • TB6612FNG: 효율이 좋고 작은 로봇에 많이 사용

아두이노 연결 방법

아래 연결 예시는 L298N 모터 드라이버를 기준으로 합니다.

L298N 핀 아두이노 연결 핀 역할
IN1 D8 모터 방향 제어 1
IN2 D9 모터 방향 제어 2
ENA D10 모터 속도 제어 PWM
OUT1, OUT2 DC 모터 모터 연결
12V / VCC 외부 전원 + 모터 전원 공급
GND 아두이노 GND, 외부 전원 - 공통 접지
외부 전원을 사용할 때는 아두이노 GND와 모터 드라이버 GND를 반드시 연결해야 합니다.

아두이노 예시 코드

아래 코드는 DC 모터를 정방향으로 회전시키고, 잠시 멈춘 뒤, 역방향으로 회전시키는 예시입니다.

Arduino Code 
#include <Arduino.h>

const int in1 = 8;
const int in2 = 9;
const int enA = 10;

void setup() {
  pinMode(in1, OUTPUT);
  pinMode(in2, OUTPUT);
  pinMode(enA, OUTPUT);
}

void loop() {
  // 정방향 회전
  digitalWrite(in1, HIGH);
  digitalWrite(in2, LOW);
  analogWrite(enA, 180);
  delay(2000);

  // 정지
  digitalWrite(in1, LOW);
  digitalWrite(in2, LOW);
  analogWrite(enA, 0);
  delay(1000);

  // 역방향 회전
  digitalWrite(in1, LOW);
  digitalWrite(in2, HIGH);
  analogWrite(enA, 180);
  delay(2000);

  // 정지
  digitalWrite(in1, LOW);
  digitalWrite(in2, LOW);
  analogWrite(enA, 0);
  delay(1000);
}

사용할 때 주의할 점

  • DC 모터를 아두이노 핀에 직접 연결하지 않습니다.
  • 모터 전원은 가능하면 외부 전원을 사용합니다.
  • 외부 전원과 아두이노의 GND는 반드시 같이 연결해야 합니다.
  • PWM 값은 보통 0부터 255까지 사용합니다.
  • 모터가 갑자기 돌면서 큰 전류를 사용할 수 있으므로 전원 용량을 확인해야 합니다.
