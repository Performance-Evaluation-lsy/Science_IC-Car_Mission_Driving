4핀 RGB LED 사용 방법

4핀 RGB LED는 빨강, 초록, 파랑 LED가 하나의 부품 안에 들어 있는 LED입니다. 각 색의 밝기를 조절하면 빨강, 초록, 파랑뿐만 아니라 노랑, 보라, 하늘색, 흰색 같은 다양한 색을 만들 수 있습니다.

4핀 RGB LED란?

RGB LED는 Red, Green, Blue 세 가지 색을 가진 LED입니다. 4개의 핀 중 3개는 각각 빨강, 초록, 파랑을 제어하는 핀이고, 나머지 1개는 공통으로 연결되는 핀입니다.

RGB LED의 종류

  • 공통 캐소드: 가장 긴 다리를 GND에 연결합니다.
  • 공통 애노드: 가장 긴 다리를 5V에 연결합니다.
이 예제 코드는 공통 캐소드 RGB LED 기준입니다.
공통 애노드 RGB LED를 사용할 경우 코드의 밝기 값이 반대로 동작할 수 있습니다.

아두이노 연결 방법

RGB LED 핀 아두이노 연결 역할
R 핀 저항을 거쳐 D9 빨간색 밝기 제어
G 핀 저항을 거쳐 D10 초록색 밝기 제어
B 핀 저항을 거쳐 D11 파란색 밝기 제어
공통 핀 GND 공통 캐소드 기준 접지
R, G, B 핀에는 각각 220Ω ~ 330Ω 저항을 연결하는 것이 좋습니다. 저항 없이 연결하면 LED나 아두이노 핀이 손상될 수 있습니다.

RGB LED 색상 변경 예시 코드

아래 코드는 RGB LED를 빨강, 초록, 파랑, 노랑, 보라, 하늘색, 흰색 순서로 켜는 예시입니다.

Arduino Code 
#include <Arduino.h>

const int redPin = 9;
const int greenPin = 10;
const int bluePin = 11;

void setColor(int red, int green, int blue) {
  analogWrite(redPin, red);
  analogWrite(greenPin, green);
  analogWrite(bluePin, blue);
}

void setup() {
  pinMode(redPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
  pinMode(bluePin, OUTPUT);
}

void loop() {
  setColor(255, 0, 0);
  delay(1000);

  setColor(0, 255, 0);
  delay(1000);

  setColor(0, 0, 255);
  delay(1000);

  setColor(255, 255, 0);
  delay(1000);

  setColor(255, 0, 255);
  delay(1000);

  setColor(0, 255, 255);
  delay(1000);

  setColor(255, 255, 255);
  delay(1000);

  setColor(0, 0, 0);
  delay(1000);
}

코드 설명

  • redPin: 빨간색 LED를 제어하는 핀입니다.
  • greenPin: 초록색 LED를 제어하는 핀입니다.
  • bluePin: 파란색 LED를 제어하는 핀입니다.
  • analogWrite(): PWM 신호로 LED 밝기를 조절합니다.
  • setColor(255, 0, 0): 빨간색만 최대 밝기로 켭니다.
  • setColor(0, 0, 0): 모든 색을 꺼서 LED를 끕니다.

색상 값 예시

색상 Red Green Blue
빨강 255 0 0
초록 0 255 0
파랑 0 0 255
노랑 255 255 0
보라 255 0 255
흰색 255 255 255

사용할 때 주의할 점

  • RGB LED가 공통 캐소드인지 공통 애노드인지 확인해야 합니다.
  • 각 색상 핀에는 저항을 따로 연결하는 것이 안전합니다.
  • 아두이노 UNO에서 밝기 조절을 하려면 PWM 핀을 사용해야 합니다.
  • 공통 애노드 RGB LED는 공통 핀을 5V에 연결합니다.
  • 공통 애노드 방식에서는 값이 작을수록 밝고, 값이 클수록 어두워질 수 있습니다.
