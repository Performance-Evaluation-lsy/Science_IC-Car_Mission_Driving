진동 모터 사용 방법

진동 모터는 전류가 흐르면 빠르게 회전하면서 진동을 만드는 모터입니다. 스마트폰 알림, 게임 컨트롤러, 웨어러블 기기, 경고 장치, 터치 피드백 장치 등에 자주 사용됩니다.

진동 모터란?

진동 모터는 작은 DC 모터의 축에 무게 중심이 한쪽으로 치우친 추가 달려 있는 부품입니다. 모터가 회전하면 중심이 흔들리면서 진동이 발생합니다.

동작 원리

  • 모터에 전류가 흐르면 내부 코일과 자석의 힘으로 축이 회전합니다.
  • 축에 달린 추가 한쪽으로 치우쳐 있기 때문에 회전할 때 흔들림이 생깁니다.
  • 이 흔들림이 진동으로 느껴집니다.
  • PWM 신호를 사용하면 진동 세기를 어느 정도 조절할 수 있습니다.
진동 모터는 작은 부품이지만 모터이기 때문에 아두이노 핀에 직접 연결하지 않는 것이 좋습니다.
보통 트랜지스터나 모터 드라이버를 사용해 제어합니다.

아두이노 연결 방법

아래 연결 예시는 NPN 트랜지스터를 사용해 진동 모터를 제어하는 방식입니다.

부품 / 핀 연결 위치 역할
진동 모터 + 5V 모터 전원 공급
진동 모터 - 트랜지스터 Collector 모터 전류 제어
트랜지스터 Emitter GND 접지
트랜지스터 Base 저항을 거쳐 D9 아두이노 제어 신호 입력
다이오드 모터 양쪽에 병렬 연결 역전압 보호
트랜지스터 Base에는 보통 220Ω ~ 1kΩ 저항을 연결합니다.
모터에서 생기는 역전압을 줄이기 위해 플라이백 다이오드를 함께 사용하는 것이 안전합니다.

아두이노 예시 코드

아래 코드는 진동 모터를 1초 동안 켜고, 1초 동안 끄는 동작을 반복하는 예시입니다.

Arduino Code 
#include <Arduino.h>

const int motorPin = 9;

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  digitalWrite(motorPin, HIGH);
  delay(1000);

  digitalWrite(motorPin, LOW);
  delay(1000);
}

PWM으로 진동 세기 조절하기

PWM 핀을 사용하면 진동 모터의 세기를 조절할 수 있습니다. 아두이노 UNO에서는 0부터 255 사이의 값을 사용합니다.

Arduino PWM Code 
#include <Arduino.h>

const int motorPin = 9;

void setup() {
  pinMode(motorPin, OUTPUT);
}

void loop() {
  analogWrite(motorPin, 80);
  delay(1000);

  analogWrite(motorPin, 180);
  delay(1000);

  analogWrite(motorPin, 255);
  delay(1000);

  analogWrite(motorPin, 0);
  delay(1000);
}

사용할 때 주의할 점

  • 진동 모터를 아두이노 핀에 직접 연결하지 않는 것이 좋습니다.
  • 모터 전류가 부족하면 진동이 약하거나 동작하지 않을 수 있습니다.
  • 트랜지스터, 저항, 다이오드를 함께 사용하면 회로를 더 안전하게 만들 수 있습니다.
  • 진동 모터를 오래 켜두면 발열이 생길 수 있습니다.
  • ESP32에서 사용할 때는 PWM 제어 방식이 아두이노 UNO와 다를 수 있습니다.
