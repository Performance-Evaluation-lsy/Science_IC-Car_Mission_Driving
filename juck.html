적외선 센서 사용 방법

적외선 센서는 눈에 보이지 않는 적외선을 이용해 물체를 감지하는 센서입니다. 아두이노, ESP32 같은 보드에서 장애물 감지, 라인트레이서, 거리 감지, 자동문, 로봇 회피 장치 등에 자주 사용됩니다.

적외선 센서란?

적외선 센서는 적외선 LED에서 빛을 내보낸 뒤, 물체에 반사되어 돌아오는 적외선을 수신부에서 감지하는 센서입니다. 물체가 가까이 있으면 반사되는 적외선이 많아져 감지 신호가 발생합니다.

동작 원리

  • 적외선 LED가 적외선을 발사합니다.
  • 적외선이 물체에 부딪혀 반사됩니다.
  • 수신부가 반사된 적외선을 감지합니다.
  • 센서가 물체 감지 여부를 디지털 신호로 출력합니다.
적외선 장애물 감지 센서는 보통 디지털 출력을 사용합니다.
대부분의 모듈은 물체가 감지되면 LOW 값을 출력하는 경우가 많습니다.

아두이노 연결 방법

적외선 센서 핀 아두이노 연결 핀 역할
VCC 5V 전원 공급
GND GND 접지
OUT D2 물체 감지 신호 출력
센서 모듈에 있는 가변저항을 돌리면 감지 거리나 민감도를 조절할 수 있습니다.

아두이노 예시 코드

아래 코드는 적외선 센서가 물체를 감지하면 LED를 켜고, 감지하지 않으면 LED를 끄는 예시입니다.

Arduino Code 
#include <Arduino.h>

const int irSensorPin = 2;
const int ledPin = 13;

int sensorValue = 0;

void setup() {
  Serial.begin(9600);

  pinMode(irSensorPin, INPUT);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  sensorValue = digitalRead(irSensorPin);

  if (sensorValue == LOW) {
    digitalWrite(ledPin, HIGH);
    Serial.println("Object Detected");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("No Object");
  }

  delay(300);
}

코드 설명

  • digitalRead(irSensorPin): 적외선 센서의 출력값을 읽습니다.
  • LOW: 물체가 감지된 상태로 사용하는 경우가 많습니다.
  • HIGH: 물체가 감지되지 않은 상태로 사용하는 경우가 많습니다.
  • digitalWrite(ledPin, HIGH): LED를 켭니다.
  • Serial.println(): 감지 상태를 시리얼 모니터에 출력합니다.

사용할 때 주의할 점

  • 검은색 물체는 적외선을 잘 반사하지 않아 감지가 약할 수 있습니다.
  • 강한 햇빛이나 외부 적외선 때문에 오작동할 수 있습니다.
  • 센서의 감지 거리는 모듈 종류와 가변저항 설정에 따라 달라집니다.
  • 물체의 색상, 재질, 각도에 따라 감지 정확도가 달라질 수 있습니다.
  • ESP32에서 사용할 때는 센서 출력 전압이 3.3V 입력에 안전한지 확인해야 합니다.
