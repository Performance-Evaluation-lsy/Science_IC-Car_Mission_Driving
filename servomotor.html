서보 모터 사용 방법

서보 모터는 원하는 각도로 회전시킬 수 있는 모터입니다. 아두이노, ESP32 같은 보드에서 로봇 팔, 자동차 조향 장치, 자동문, 카메라 방향 제어 등에 자주 사용됩니다.

서보 모터란?

서보 모터는 일반 DC 모터처럼 계속 회전하는 것이 아니라, 사용자가 지정한 각도로 움직일 수 있는 모터입니다. 보통 0도부터 180도까지 회전할 수 있는 서보 모터가 많이 사용됩니다.

동작 원리

  • 보드에서 서보 모터의 신호선으로 제어 신호를 보냅니다.
  • 서보 모터는 받은 신호를 기준으로 목표 각도를 정합니다.
  • 내부 회로와 기어가 움직이면서 축을 해당 각도로 회전시킵니다.
  • 지정한 각도에 도달하면 그 위치를 유지합니다.
서보 모터는 보통 PWM 신호를 이용해 각도를 제어합니다.
아두이노에서는 Servo 라이브러리를 사용하면 쉽게 제어할 수 있습니다.

아두이노 연결 방법

서보 모터 선 아두이노 연결 핀 역할
빨간색 선 5V 전원 공급
갈색 / 검은색 선 GND 접지
주황색 / 노란색 선 D9 제어 신호
서보 모터가 많이 흔들리거나 힘이 부족하면 아두이노의 5V 핀 대신 외부 전원을 사용하는 것이 좋습니다. 이때 외부 전원의 GND와 아두이노 GND는 반드시 연결해야 합니다.

아두이노 예시 코드

아래 코드는 서보 모터를 0도, 90도, 180도로 차례대로 움직이는 예시입니다.

Arduino Code 
#include <Arduino.h>
#include <Servo.h>

Servo myServo;

const int servoPin = 9;

void setup() {
  myServo.attach(servoPin);
}

void loop() {
  myServo.write(0);
  delay(1000);

  myServo.write(90);
  delay(1000);

  myServo.write(180);
  delay(1000);
}

사용할 때 주의할 점

  • 서보 모터는 일반적으로 0도부터 180도까지만 회전합니다.
  • 무리하게 회전 범위를 넘기면 모터가 손상될 수 있습니다.
  • 전류가 부족하면 모터가 떨리거나 제대로 움직이지 않을 수 있습니다.
  • 여러 개의 서보 모터를 사용할 때는 외부 전원 사용을 권장합니다.
  • ESP32에서 사용할 때는 Servo 라이브러리 대신 ESP32Servo 라이브러리를 사용하는 경우가 많습니다.
