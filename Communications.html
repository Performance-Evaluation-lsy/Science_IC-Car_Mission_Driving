초음파 센서 사용 방법

초음파 센서는 소리를 이용해 물체와의 거리를 측정하는 센서입니다. 아두이노, ESP32 같은 보드에서 장애물 감지, 거리 측정, 자동차 후방 감지, 로봇 회피 시스템 등에 자주 사용됩니다.

초음파 센서란?

초음파 센서는 사람이 들을 수 없는 높은 주파수의 소리를 내보낸 뒤, 그 소리가 물체에 반사되어 돌아오는 시간을 측정해서 거리를 계산합니다.

동작 원리

  • Trig 핀에서 초음파 신호를 발생시킵니다.
  • 초음파가 물체에 닿고 다시 센서로 돌아옵니다.
  • Echo 핀에서 돌아오는 데 걸린 시간을 측정합니다.
  • 측정한 시간을 이용해서 거리를 계산합니다.
거리 계산 공식: 거리(cm) = 시간 × 0.0343 ÷ 2
2로 나누는 이유는 초음파가 물체까지 갔다가 다시 돌아오기 때문입니다.

아두이노 연결 방법

초음파 센서 핀 아두이노 연결 핀 역할
VCC 5V 전원 공급
GND GND 접지
Trig D9 초음파 발사 신호
Echo D10 초음파 반사 신호 측정

아두이노 예시 코드

아래 코드는 초음파 센서로 거리를 측정한 뒤, 시리얼 모니터에 거리 값을 cm 단위로 출력하는 예시입니다.

Arduino Code 
#include <Arduino.h>

const int trigPin = 9;
const int echoPin = 10;

long duration;
float distance;

void setup() {
  Serial.begin(9600);

  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

void loop() {
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH);

  distance = duration * 0.0343 / 2;

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  delay(500);
}

사용할 때 주의할 점

  • 센서 앞에 너무 가까운 물체가 있으면 정확도가 떨어질 수 있다.
  • 천, 스펀지처럼 소리를 잘 흡수하는 물체는 측정이 부정확할 수 있다.
  • 센서가 비스듬히 물체를 바라보면 초음파가 제대로 돌아오지 않을 수 있다.
  • ESP32에서 사용할 때는 Echo 핀의 5V 신호를 3.3V로 낮춰야 안전하다.
