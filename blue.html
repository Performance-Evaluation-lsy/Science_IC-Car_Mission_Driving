블루투스 통신 모듈 사용 방법

블루투스 통신 모듈은 아두이노와 스마트폰, 컴퓨터 같은 장치가 무선으로 데이터를 주고받을 수 있게 해주는 모듈입니다. 대표적으로 HC-05, HC-06 모듈이 많이 사용됩니다.

블루투스 통신 모듈이란?

블루투스 통신 모듈은 근거리 무선 통신을 위한 부품입니다. 아두이노에서는 주로 시리얼 통신 방식으로 블루투스 모듈과 데이터를 주고받습니다.

대표적인 블루투스 모듈

  • HC-05: 마스터 모드와 슬레이브 모드를 모두 지원합니다.
  • HC-06: 보통 슬레이브 모드만 지원합니다.
  • HM-10: BLE 통신을 지원하는 모듈입니다.
아두이노에서 HC-05, HC-06은 보통 UART 시리얼 통신으로 제어합니다.
즉, TX와 RX 핀을 이용해 데이터를 주고받습니다.

동작 원리

  • 스마트폰이 블루투스 모듈과 페어링됩니다.
  • 스마트폰 앱에서 문자를 보내면 블루투스 모듈이 데이터를 받습니다.
  • 블루투스 모듈은 받은 데이터를 아두이노로 전달합니다.
  • 아두이노는 받은 값에 따라 LED, 모터, 부저 등을 제어할 수 있습니다.

아두이노 연결 방법

아래 연결 예시는 HC-05 블루투스 모듈을 기준으로 합니다.

블루투스 모듈 핀 아두이노 연결 핀 역할
VCC 5V 전원 공급
GND GND 접지
TXD D2 블루투스 모듈이 보내는 데이터 수신
RXD D3 블루투스 모듈로 데이터 전송
연결할 때는 TXD는 아두이노 RX 역할 핀에, RXD는 아두이노 TX 역할 핀에 연결합니다.
즉, TX와 RX는 서로 교차해서 연결합니다.

아두이노 예시 코드

아래 코드는 스마트폰에서 문자 1을 보내면 LED를 켜고, 문자 0을 보내면 LED를 끄는 예시입니다.

Arduino Code 
#include <Arduino.h>
#include <SoftwareSerial.h>

SoftwareSerial bluetooth(2, 3);

const int ledPin = 13;
char data;

void setup() {
  Serial.begin(9600);
  bluetooth.begin(9600);

  pinMode(ledPin, OUTPUT);

  Serial.println("Bluetooth Ready");
}

void loop() {
  if (bluetooth.available()) {
    data = bluetooth.read();

    Serial.print("Received: ");
    Serial.println(data);

    if (data == '1') {
      digitalWrite(ledPin, HIGH);
    } else if (data == '0') {
      digitalWrite(ledPin, LOW);
    }
  }
}

코드 설명

  • SoftwareSerial bluetooth(2, 3): D2를 RX, D3을 TX처럼 사용합니다.
  • bluetooth.begin(9600): 블루투스 통신 속도를 9600bps로 설정합니다.
  • bluetooth.available(): 블루투스로 받은 데이터가 있는지 확인합니다.
  • bluetooth.read(): 받은 데이터를 하나 읽습니다.
  • data == '1': 스마트폰에서 1을 보냈는지 확인합니다.

사용할 때 주의할 점

  • TX와 RX는 서로 교차해서 연결해야 합니다.
  • HC-05, HC-06의 기본 통신 속도는 보통 9600bps입니다.
  • 스마트폰과 연결하려면 먼저 블루투스 페어링을 해야 합니다.
  • HC-05의 기본 비밀번호는 보통 1234 또는 0000입니다.
  • ESP32는 블루투스 기능이 내장되어 있어 외부 HC-05 모듈이 필요 없는 경우도 있습니다.
